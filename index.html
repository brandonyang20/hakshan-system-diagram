05 — n8n Automation Engine
N8N-CORE
n8n — Central Orchestration Engine
Receives all inputs. Runs Claude OCR. Routes data. Writes to Google Sheets. Sends WhatsApp outputs. Runs scheduled jobs.
Claude OCR pipeline
Reads invoice image/PDF from WhatsApp · Extracts: supplier name, document type (SO/QT/IV), invoice number, date, line items, amounts · Confidence check on each field
Company name attribution
Billing company name matched against 8 outlet Sdn Bhd names · Routes to correct outlet Google Sheet + Master Ledger tab · No group ID config needed
Supplier auto-detection
Checks extracted supplier name vs Supplier Master tab · If not found → auto-creates new row + new outlet sheet tab → WhatsApp alert to admin for verification
Dual write logic
Every invoice simultaneously writes to: (1) Outlet Google Sheet → correct supplier tab → appends row, and (2) Master Ledger Invoice Log tab → tagged by outlet + supplier
Price change tracking
Extracted unit price vs last recorded price in SKU Master · If change >5% → trigger spike alert WhatsApp · Update live price table with timestamp
Scheduled jobs
11pm — FeedMe API pull · Saturday — weekly snapshot archive · Saturday — draft inter-co statement · Monday 8am — weekly report to owner

Analyses per-outlet historical data to recommend order quantities before each 3-day ordering cycle. Learns from manager overrides over time.

Output and learning
